WP H-PHP-Widget

== Description ==
Dieses Plugin fügt ein Widget zum Einfügen von PHP-Code im Dashboard hinzu.

== Installation ==
1. Upload `wp-h-tools  to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
